# Gestor Académico 

##  Objetivo y Funcionalidad
Este proyecto es una herramienta de uso interno para la gestión, cálculo y seguimiento de calificaciones académicas de forma privada. 

El sistema permite automatizar el proceso de evaluación, aplicando ponderaciones específicas a cada actividad para obtener promedios reales en tiempo real.

###  Capacidades del CRUD:
* **Estudiantes:** Registrar, listar, modificar datos y dar de baja alumnos del curso.
* **Evaluaciones:** Ingresar notas individuales vinculadas a porcentajes configurables
* **Cálculo Automático:** Procesamiento de promedios finales basados en el peso de cada nota.
* **Ajustes:** Modificación inmediata de calificaciones y actualización instantánea del estado académico.

---

##  Arquitectura del Sistema
* `/frontend`: Interfaz para la visualización de promedios y carga de notas.
* `/backend`: Lógica de cálculo de porcentajes y gestión de registros.
* `/database`: Tablas de Estudiantes, Notas y Configuración de Ponderaciones.
